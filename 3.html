Предупреждения

Утилита генерирует 2 простых текстовых файла: TableValues.txt и Coeffs.txt. Это таблица распознаваемых длин волн и коэффициенты для расчета температурных кривых соответственно. Их можно менять ТОЛЬКО не нарушая структуры.

В случае возникновения ошибок при загрузке утилиты, настроечные файлы следует удалить.

Рекомендуется добавлять и изменять данные через программу, а не напрямую через файлы настроек.


Если загружено слишком много спектров - изображение оставит желать лучшего даже в плоском режиме

Если спектры не захотели грузиться и вылез Access Violation - значит, используется нестабильная бета, стоит попробовать пересохранить спектры в другой файл из ПО к монохроматору, возможно, с меньшим числом спектров



В модуле обработки ВАХ следует загружать ранее пересохраненные при предыдущем открытии данные иначе, чем "сырые" экспериментальные - файлы имеют разный формат.

При дифференцирующей фильтрации кривых методом Савицкого-Голая следует помнить, что полуширина окна (250 точек) с каждого края не обрабатывается, необходимо значительное число значащих точек
